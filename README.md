## Features

    <li> Run, Attack, Idle and Death Animation </li>
    <li> Health Bar </li>
    <li> Timer </li>
</ul>

 ## Controls
  ### Player 1:
    W : Jump
    A : Move Left
    D : Move Right
    Space : Attack
    
  ### Player 2:
    Up Arrow : Jump
    Left Arrow  : Move Left
    Right Arrow  : Move Right
    Down Arrow  : Attack
